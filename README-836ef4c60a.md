## 快速开始

### 克隆仓库
```bash
git clone https://github.com/你的用户名/你的用户名.github.io.git
cd 你的用户名.github.io
```

### 本地预览
```bash
# 安装依赖
bundle install

# 启动本地服务器
bundle exec jekyll serve
```

访问 http://localhost:4000 预览博客

### 发布新文章
1. 在 `_posts/` 目录创建文件：`YYYY-MM-DD-文章标题.md`
2. 编写文章内容（使用Markdown格式）
3. 提交并推送到GitHub
4. 等待GitHub Pages自动部署
